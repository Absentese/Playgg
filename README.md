# playgg

Интернет-магазин **цифровых видеоигр** на **Laravel 13**: каталог ключей, корзина, заказы, профиль, AI-чат поддержки и админ-панель.

Дипломный проект: разработка информационной системы веб-приложения интернет-магазина цифровых видеоигр.

## Требования

- PHP 8.3+
- Composer
- SQLite (по умолчанию) или MySQL

## Установка и запуск

```bash
cd gamevault
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
```

Откройте: http://127.0.0.1:8000

## Функциональность

- Главная: хиты продаж, категории, лучшие скидки
- Каталог с фильтрами (категория, скидки, предзаказы, сортировка)
- Карточка игры, корзина, оформление и оплата (имитация)
- Регистрация, профиль, история заказов

- Страницы: доставка ключей, гарантия, пополнение Steam
- Админ-панель: товары, заказы, пользователи, обращения

## Стек

- Laravel 13, Blade, Bootstrap 5, SQLite
- Дизайн: тёмная игровая тема (фиолетово-циановая палитра)
